Integration with Apache Lucene 2.4. This extension provides a client connector for Lucene Solr 1.3 sub-project (complete search and indexing engine) that can be used as an alternative to deployment within a Servlet container. There is also support for automatic extraction of metadata from Restlet representation by leveraging Lucene Tika 0.4 sub-project. @since Restlet 2.0 @see Apache Lucene
