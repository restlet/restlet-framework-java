--------------
JBoss RESTEasy 
--------------

"RESTEasy is a JBoss project that provides various frameworks to 
help you build RESTful Web Services and RESTful Java applications. 
It is a fully certified and portable implementation of the JAX-RS 
specification. JAX-RS is a new JCP specification that provides a 
Java API for RESTful Web Services over the HTTP protocol."

For more information:
http://www.jboss.org/resteasy
